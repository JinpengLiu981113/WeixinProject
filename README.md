# WeixinProject
### 数据库说明（database文件夹）
* Answer：社区回答
* Community：社区提问
* Knowledge：知识点
* Notice：公告
* QuestionBook：题目集合
* UserInfo：用户信息记录（包括签到和分数
* WrongBook：错题本

* Chapter2: 10以内减
* Chapter3: 100以内加
* Chapter4: 100以内减
* Chapter5: 乘法口诀表
* Chapter6: 除法口诀表
* Chapter7: 100以内加减混合

* Chapter10: 大数乘除法（可能用到竖式计算）
* Chapter11: 大数加减乘除混合（可能用到竖式计算）
